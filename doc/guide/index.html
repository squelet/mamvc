Quick guide to µ-AMVC

Trio - The MVC pattern

VIEW: DOM Builder

TRIO is a web MVC technology, so the presentation (VIEW) layer is at the end represented as HTML Dom. Therefore it needs to allow fully flexible way of creating and manipulating the DOM tree of the page.

µ-AMVC is pure Ecma Script 5 based library, so the API is JavaScript. For DOM manipulation it uses a builder pattern per element. Each builder instance wraps a DOMNode, and provides methods to simply and intuitively manipulate it.
